# 📋 FSD Lab 9 — Accessible Form with Validation

A React app implementing an **accessible, validated form** with real-time error feedback.

## ✨ Features
- Client-side form validation (name, email, password)
- Accessible HTML — proper labels, ARIA attributes
- Real-time error messages on invalid input
- Clean, minimal UI with focus states
- Built with React controlled components

## 🛠️ Tech Stack
- React 19
- Vite
- Vanilla CSS

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
